# 关于此固件

基于[P3TERX/Actions-OpenWrt](https://github.com/P3TERX/Actions-OpenWrt)，用Github Actions为K2P路由器自动编译固件。

## 包含软件包

主要添加了mwan3等常用插件，详见[.config](https://github.com/carrothu-cn/Actions-OpenWrt-k2p/blob/main/.config)文件配置。
